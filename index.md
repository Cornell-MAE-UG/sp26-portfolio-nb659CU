# Mechanical Advantage Design Challenge

## Overview

This project demonstrates mechanical engineering principles applied to a practical tool design. The challenge was to design a nutcracker capable of applying enough force to crack a macadamia nut based on average human grip strength.

## Key Objectives

- **Analyze** the mechanical advantage requirements to crack a macadamia nut
- **Apply** force equilibrium principles to calculate required dimensions
- **Design** a functional, ergonomic tool within practical size constraints

## Design Parameters

| Requirement | Value |
|-----------|-------|
| Nut cracking load (macadamia) | 222 kgf |
| 50th percentile female grip strength (20-24 yrs) | 28.6 kgf |
| Average nut diameter | ~25 mm |
| Maximum tool length | ~10 cm |
